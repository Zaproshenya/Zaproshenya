# План покращень для "Запрошення"

## Завдання

### 1. Уніфікація border-radius
- Привести всі картки до `--radius-card: 14px`
- Файли: `css/components.css`, `css/pages.css`, `css/dashboard.css`

### 2. Hover-ефекти для всіх карток
- Додати `translateY(-2px)` + тінь для: `.stat-card`, `.chart-card`, `.table-card`, `.complaint-card`, `.notif-item`
- Файли: `css/dashboard.css`, `css/pages.css`

### 3. Доступність (a11y)
- Додати `:focus-visible` стилі для карток та кнопок
- Файл: `css/components.css`

### 4. Скелетони СКРІЗЬ
Показувати скелетони при завантаженні на КОЖНІЙ сторінці та для КОЖНОГО елемента.

**Сторінки:**
| Сторінка | Що показувати як скелетон |
|----------|--------------------------|
| Головна (`home.js`) | Список карток запрошень (вихідні + вхідні) |
| Створення (`create.js`) | Форма з порожніми полями |
| Перегляд запрошення (`invite.js`) | Конверт з деталями |
| Друзі (`friends.js`) | Список друзів, запити, запрошення від друзів |
| Профіль (`profile.js`) | Аватар, інформація, налаштування |
| Профіль користувача (`user-profile.js`) | Аватар, ім'я, кнопки |
| Логін (`login.js`) | Форма авторизації |
| Дашборд (`dashboard.js`) | Статистика, графіки, таблиці, скарги |

**Елементи для скелетонів:**
- Текст (лінії різної довжини)
- Іконки (кола/прямокутники)
- Кнопки (прямокутники)
- Аватари (кола)
- Числа (великі прямокутники)
- Графіки (прямокутник з градієнтом)
- Таблиці (рядки з лініями)

### 5. Баг: творець може прийняти/відхилити своє запрошення
- Кнопки Accept/Decline/Reschedule будуть видимі, але **заблоковані** для творця
- Стиль: `opacity: 0.5`, `cursor: not-allowed`, `pointer-events: none`
- Кнопки стають сірими та затемненими
- Файли: `js/pages/invite.js`, `css/components.css`

### 6. Push на GitHub
- Створити коміт та запушити зміни

## CSS для скелетонів

Додати в `css/components.css`:

```css
/* Базовий скелетон-рядок */
.skeleton-line {
  height: 14px;
  background: linear-gradient(90deg, var(--warm) 25%, #e8dfd0 50%, var(--warm) 75%);
  background-size: 200% 100%;
  animation: shimmer 1.5s infinite;
  border-radius: var(--radius-sm);
}

/* Різні розміри */
.skeleton-line.w-full { width: 100%; }
.skeleton-line.w-3-4 { width: 75%; }
.skeleton-line.w-1-2 { width: 50%; }
.skeleton-line.w-1-4 { width: 25%; }

/* Скелетон-коло (аватар, іконка) */
.skeleton-circle {
  border-radius: 50%;
  background: linear-gradient(90deg, var(--warm) 25%, #e8dfd0 50%, var(--warm) 75%);
  background-size: 200% 100%;
  animation: shimmer 1.5s infinite;
}

/* Скелетон-кнопка */
.skeleton-btn {
  height: 40px;
  border-radius: var(--radius-md);
  background: linear-gradient(90deg, var(--warm) 25%, #e8dfd0 50%, var(--warm) 75%);
  background-size: 200% 100%;
  animation: shimmer 1.5s infinite;
}

/* Скелетон-картка */
.skeleton-card {
  background: var(--card);
  border: 1.5px solid var(--border);
  border-radius: var(--radius-card);
  padding: 16px 20px;
  margin-bottom: 12px;
}
```

## JS-логіка скелетонів

Кожна сторінка матиме функцію `renderSkeleton()`, яка повертає HTML скелетонів. Після завантаження даних скелетони замінюються реальним контентом.

```js
// Приклад для home.js
renderSkeleton() {
  return `
    <div class="inv-card skeleton-card">
      <div class="skeleton-circle" style="width:40px;height:40px"></div>
      <div style="flex:1">
        <div class="skeleton-line w-3-4" style="margin-bottom:8px"></div>
        <div class="skeleton-line w-1-2"></div>
      </div>
    </div>
  `.repeat(5);
}
```

## CSS для заблокованих кнопок

Додати в `css/components.css`:

```css
/* Заблоковані кнопки для творця запрошення */
.btn-disabled {
  opacity: 0.5;
  cursor: not-allowed;
  pointer-events: none;
  filter: grayscale(40%);
}
.btn-disabled:hover {
  transform: none;
  box-shadow: none;
}
```

## Верифікація

1. Відкрити index.html в браузері
2. Перейти на кожну сторінку та перевірити скелетони
3. Перевірити hover-ефекти на всіх картках
4. Перевірити табуляцію (Tab) — видимий outline
5. Створити запрошення → відкрити посилання → переконатися, що кнопки затемнені та заблоковані для творця
6. Запушити на GitHub
